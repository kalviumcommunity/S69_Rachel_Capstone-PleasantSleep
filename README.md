## Project Description
This project is a **real-time chat application** with AI-powered **live translation**.  
The main goal is to allow users to **send messages in any language**, and the recipient will receive it **translated into their preferred language** automatically.

### Features:
- User authentication (username/password & Google OAuth)
- Real-time chat messaging
- AI-powered message translation
- Choice of preferred language for each user
- MongoDB backend with JWT-based authentication
- Frontend built with React.js
- Backend built with Node.js and Express.js

## Tech Stack
- Frontend: **React.js, Vite**
- Backend: **Node.js, Express.js**
- Database: **MongoDB (Mongoose)**
- Authentication: **JWT, Google OAuth**
- Translation & AI integration: **Gemini API / AI translation API**
