# AI Integrated Multilingual Chat Application

## Project Description
This project is a real-time chat application built using the MERN stack.
It supports AI-powered live message translation using the Gemini API.
Users can select a preferred language to receive messages, and all incoming
messages are translated automatically.

## Tech Stack
- Frontend: React.js (Vite)
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT & Google OAuth
- AI Translation: Gemini API

## Features
- User authentication
- Real-time chat
- Automatic language detection
- Live translation to selected language
